\begin{kitpropiedades}{ESTRATEGIAS DE RESOLUCIÓN}
    Para enfrentar esta guía con éxito, recuerda que resolver un sistema de ecuaciones no es solo un ejercicio algebraico mecánico, sino que buscas encontrar el punto exacto $(x, y)$ donde se cruzan dos rectas, o el punto $(x, y, z)$ donde se interceptan tres planos en el espacio.

    \vspace{0.5em}
    \textbf{Métodos para sistemas de $2\times2$:}
    \begin{itemize}[itemsep=0.5em]
        \item \textbf{Reducción:} Ideal para eliminar variables sumando ecuaciones hacia abajo.
        \item \textbf{Sustitución:} Perfecto si ya tienes una letra casi despejada.
        \item \textbf{Igualación:} Útil cuando es fácil despejar la misma variable en ambas ecuaciones.
    \end{itemize}

    \vspace{0.5em}
    \textbf{Para sistemas de $3\times3$:} \\
    Tu mejor estrategia es usar la \textbf{reducción} tomando pares de ecuaciones para eliminar la misma incógnita en ambos, transformando así el problema original en un sistema de $2\times2$ que ya sabes resolver.

    \vspace{0.5em}
    \textbf{Interpretación de resultados:} \\
    Revisa siempre tus signos negativos al operar y no te asustes si durante el desarrollo llegas a un resultado extraño:
    \begin{itemize}[itemsep=0.5em]
        \item \textbf{Si llegas a una falsedad matemática (ej. $0=5$):} Esto simplemente significa que el sistema \textbf{no tiene solución} (son rectas o planos paralelos que no se tocan).
        \item \textbf{Si llegas a una identidad obvia (ej. $0=0$):} Significará que las ecuaciones representan la misma figura geométrica y, por lo tanto, el sistema tendrá \textbf{infinitas soluciones}.
    \end{itemize}
\end{kitpropiedades}
